A brief description of the plugin.

== Description ==
Hide attachments assigned to a specific category term from
the media gallery.


== Installation ==

= Manual Installation =

1. Upload the entire `/hide-attachments` directory to the `/wp-content/plugins/` directory.
2. Activate Hide Attachments through the 'Plugins' menu in WordPress.

== Settings ==

1. Select the Category Terms from the 'Settings -> Media' menu. The attachments assigned to the selected category terms will not be visible in the Media Library.
